## Context

This project currently contains a development workflow spec that is no longer in use.

## Goals / Non-Goals

**Goals:**
- Remove all development workflow spec files from the project

**Non-Goals:**
- Do not add new workflows or replace the removed one
- Do not refactor other parts of the codebase unrelated to this removal

## Decisions

1. **Removal scope**: Identify and remove all files directly associated with the development workflow. This includes only openspec specs.
